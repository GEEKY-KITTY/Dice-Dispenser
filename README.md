# 🎲 DnD Dice Dispenser

A fully 3D-printable dice dispenser designed for tabletop RPGs like **Dungeons & Dragons**, Pathfinder, and other dice-heavy games.

This project combines functional mechanical design with tabletop aesthetics to create a compact and satisfying way to dispense dice during gameplay.

---

## ✨ Features

- Fully 3D printable
- Optimized for FDM printing
- Smooth dice dispensing mechanism
- Compact tabletop-friendly design
- Modular and easy to assemble
- Works with standard RPG dice sets
- Customizable for different themes and colors



---

## 🛠️ Printing Details

### Recommended Settings

| Setting | Value |
|---|---|
| Layer Height | 0.2 mm |
| Infill | 15–25% |
| Supports | Depends on part |
| Material | PLA / PETG |
| Nozzle | 0.4 mm |

### Recommended Orientation

- Print the main body upright
- Smaller moving parts may require supports
- Use brim if bed adhesion is weak
